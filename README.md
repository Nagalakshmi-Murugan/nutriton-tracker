# Nutrition Tracker

A simple C++ console-based Nutrition Tracker application that allows users to manage food entries and track daily nutritional intake.

## Features

- Add food items
- View all food entries
- Delete food entries
- Calculate total calories, protein, carbs, and fat
- Daily calorie goal tracking
- Detect when calorie goal is exceeded

## Technologies Used

- C++
- STL Vector
- Object-Oriented Programming (OOP)

## Project Structure

- Food class for storing nutrition data
- Vector for managing food entries
- Menu-driven interface

## Future Improvements

- Edit food entries
- Save data to file
- Multi-day tracking
- Analytics dashboard
- GUI/Web version

## How to Run

1. Compile:

```bash
g++ main.cpp -o nutrition_tracker
```

2. Run:

```bash
./nutrition_tracker
```

## Sample Menu

```text
1. Add Food
2. Display Foods
3. Calculate Total Nutrition
4. Delete Food
```
